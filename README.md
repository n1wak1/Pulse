1) Приложение, в котором пользователи смогут отслеживать свои задачи. Pulse позволяет:
   
   • создавать и распределять задачи между участниками

   • следить за прогрессом по спринтам через Kanban-доску
 
   • проводить ежедневные стендапы прямо в приложении

2) Тип проекта: Мобильное приложение

3) Цель проекта: Создать приложение с удобным интерфейсом для распределения и отслеживания задач по грейдам.
   
4) Технологии:
   
   • Front-end: Flutter 3
  
   • Back-end: Java 21 (Spring Boot 3, REST API)

   • База данных: PostgreSQL 16
 
   • Авторизация и хранилище: Firebase Auth
 
   • Аналитика: Apache Superset / собственные графики

   • Инфраструктура: Docker + VPS server.
